# App Store checklist

Before submitting, prepare these items:

## App Store Connect

- App name: Typing Challenge
- Subtitle
- Description
- Keywords
- Support URL
- Marketing URL, optional
- Privacy policy URL, recommended
- Category: Games or Education
- Age rating

## Assets

- App icon: 1024 x 1024 PNG
- iPhone screenshots
- iPad screenshots if supporting iPad
- Optional preview video

## Technical checks

- Runs on a real iPhone
- Text input works with iOS keyboard
- Buttons are touch-friendly
- Login/sign-up works
- Leaderboard saves after closing/reopening the app
- No copyrighted music, fonts, or images
- No crash when internet is unavailable
- No desktop-only files included

## Review note suggestion

This is an offline educational typing challenge game. It stores local user accounts and scores only inside the app sandbox.
